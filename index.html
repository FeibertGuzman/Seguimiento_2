El Desafío Global

La contaminación del aire es uno de los principales problemas ambientales a nivel mundial. El índice AQI (Air Quality Index) actúa como el termómetro de nuestra atmósfera, midiendo contaminantes críticos como el PM2.5, PM10, NO2, y Ozono.

Este proyecto no solo visualiza el problema, va un paso más allá utilizando lógica prescriptiva para brindar recomendaciones accionables basadas en la calidad del aire de metrópolis globales como Medellín, Bogotá, Cali, México y Shanghai.

Fichas de Paso

El proceso técnico construido dentro del análisis de datos.

01

Conexión a la API (WAQI)

Utilizando el servicio en tiempo real del World Air Quality Index, nos conectamos vía peticiones HTTP seguras para obtener los niveles de contaminación actuales de ciudades específicas.

02

Creación de DataFrames

Se procesa el archivo JSON devuelto por la API. Transformamos arreglos complejos de diccionarios en estructuras tabulares altamente eficientes utilizando la librería Pandas.

03

Clasificación del AQI

Mediante reglas lógicas, convertimos el valor numérico en categorías estandarizadas globales:

  • Buena (0-50)
  • Moderada (51-100)
  • Muy Dañina (+200)
04

Modelo Prescriptivo

Donde ocurre la magia de la Inteligencia de Negocios. El sistema asocia la alerta ambiental actual con una acción recomendada automatizada para entidades de salud pública o usuarios.

05

Visualización Dinámica

El análisis culmina renderizando interfaces gráficas impactantes, mapas térmicos en vivo, y gráficos de barras comparativos para proveer contexto inmediato al tomador de decisiones.

06

Despliegue Interactivo

Todo el ecosistema se unifica e implementa en una aplicación Web a través de Streamlit, entregando analítica en un producto final al usuario.
